### Langkah 1: Buka File Workflow di GitHub

**Link:** https://github.com/Lfridyans/injourneyairports/blob/main/.github/workflows/deploy.yml

1. Klik link di atas
2. File workflow akan terbuka

### Langkah 2: Edit File (Buat Perubahan Kecil)

1. Di halaman file, klik tombol **"Edit"** (ikon pensil di kanan atas)
2. Scroll ke bawah sedikit
3. **Tambah 1 spasi** atau **tambah komentar** di baris manapun
   - Contoh: tambah komentar `# Workflow for GitHub Pages` di baris pertama

### Langkah 3: Commit Perubahan

1. Scroll ke bagian bawah halaman edit
2. Di bagian **"Commit changes"**:
   - **Title**: `Update workflow file`
   - **Description**: (kosongkan atau isi jika mau)
3. Klik tombol hijau **"Commit changes"**

### Langkah 4: Workflow Akan Muncul!

Setelah commit:

1. Buka tab **Actions**: https://github.com/Lfridyans/injourneyairports/actions
2. Workflow **"Deploy to GitHub Pages"** akan muncul!
3. Workflow bahkan akan otomatis trigger karena ada commit baru

### Langkah 5: Monitor Progress

1. Klik workflow yang muncul
2. Monitor build progress
3. Tunggu hingga status menjadi **"Success"**
